Здравствуйте, уважаемый преподаватель!
Данная программа предназначена для составления заданий для студентов по теме «Неопределенные интегралы». Далее дано краткое пособие по использованию. Приступим к ознакомлению.

1) Открыв программу, Вы видите различные задания с различными уровнями сложности/схемами (также некоторыми примерами к ним).
2) Требуется выбрать виды интегралов, которые хотите вывести в работе студента. Чтобы это сделать нужно указать уровень сложности/схему (указать галочкой) и количество уравнений этой сложности (вписать число в поле справа). ЗАМЕЧАНИЕ: после выбора вида – нажатия «галочки», можно сразу вводить количество, т.к. курсор ставится в ячейку автоматически.
3) После этого требуется нажать кнопку создать в конце страницы.
4) Нажимая на кнопку, система зашифрует желаемые задания, указанные Вами, который Вы предоставите студенту. Зашифрованный код появится на странице. Для того, чтобы скопировать его, требуется нажать комбинацию клавиш ctrl+c (клавиша ctrl находится в левом нижнем углу раскладки). Для того, чтобы вставить скопированный код в требуемое место, необходимо нажать комбинацию клавиш ctrl+v (где v – буква из английской раскладки).
5) Осталось предоставить полученный код студенту.
* Чтобы очистить все поля и приступить к созданию следующего задания, достаточно обновить страницу.
Вот и весь алгоритм действий, требуемых для создания задания для студента. С уважением, команда разработчиков.
